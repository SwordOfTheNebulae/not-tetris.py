# its not tetris i sware

~~tetris~~ tetanus on the command line, in python

### controls: 
 - w/up/x: rotate clockwise
 - a/left: move left
 - s/down: soft drop
 - d/right: move right
 - z/lctrl: rotate anticlockwise
 - c/shift: hold
 - space: hard drop
 - esc/f1: pause
 - del: exit
